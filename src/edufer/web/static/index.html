Proposal Alignment

  • Target stakeholders: EdTech product managers, instructors, administrators, and students.
  • Emotion states in scope: boredom, engagement, confusion, and frustration.
  • Goal: convert computer vision outputs into interpretable, actionable classroom signals.

Current Demo Scope

  • Browser webcam capture on a single page.
  • OpenCV + YOLO face detection on backend frames.
  • Pluggable classification module prepared for future custom DL integration.

Engagement States

Boredom Engagement Confusion Frustration

The production model should eventually distinguish all four states. Right now, the demo always emits a placeholder bored result after a face is found.

Ethics and Privacy

  • Use model output as decision support, not as a definitive judgment of a student.
  • Real deployment should require consent, retention policies, and transparent disclosure.
  • Bias checks matter because engagement cues vary across people and contexts.

Runtime Notes
